## 🚀 Getting Started  

### 👥 Clone the Repository  
```sh
git clone https://github.com/Aamir217/portfolio-website.git
cd portfolio-website
```

### 💪 Install Dependencies  
```sh
npm install
```

### 🔥 Start Development Server  
```sh
npm run dev  # For Vite
```

### 🏢 Build for Production  
```sh
npm run build
```
